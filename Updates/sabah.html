ALSABAH is a helper language that

allows you to directly access a stack and registers in // ALNOOR

and the ALSABAH REPL and interpreter. These can both be found in the // ALSABAH repository.

If you noticed, ALSABAH and ALNOOR have similar names. While I was brainstorming ideas, I remembered that in Arabic

the greeting is "Sabah Al-Khayr" and the reply is "Sabah Al-Noor". So it was a bit of word play.

It is currently in development and will take a long time to be implemented but will be very important to the language.
